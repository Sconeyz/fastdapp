# Buy/Sell tokens (Uniswap)
This component is used to embed a trading box on your page.

## Props

* `defaultInputAmount`
* `defaultOutputTokenAddress`

## Example

```
<Uniswap
    defaultInputAmount={1}
    defaultOutputTokenAddress={tokenAddress}  />
```
